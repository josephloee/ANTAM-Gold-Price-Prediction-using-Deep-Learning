# ANTAM-Gold-Price-Prediction-using-Deep-Learning
Prediksi tren harga saham harian PT Aneka Tambang Tbk (ANTM) menggunakan model Deep Learning Long Short-Term Memory (LSTM). Proyek ini membandingkan rasio split data untuk akurasi maksimal dan dilengkapi dengan antarmuka Web App interaktif berbasis Streamlit.
